## [简介](https://zh.m.wikipedia.org/wiki/OceanBase)
概述，历史详见链接。

+ 创始人：阳振坤
+ 编程语言：C
+ OceanBase(OB)是阿里巴巴集团研发的数据库软件，用于淘宝网和诸多阿里集团的云服务、部分政府机构、银行，擅长于海量资料处理，目前速度为全球排名第一。
+ 2019年，双十一支付宝自研分布式数据库Oceanbase每秒处理峰值6100万次，天猫双十一全球狂欢节全天成交额2684亿
